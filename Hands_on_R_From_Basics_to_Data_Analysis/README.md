## 📘 Overview of the Course
This course introduces fundamental R programming concepts, including vector creation, subsetting, file handling, descriptive and inferential statistics, data visualization, and linear modeling. It covers multiple projects, including gene data analysis and blood pressure statistical analysis

---

## 📚 Modules & Topics Covered
| Module | Lectures / Project | Key Focus |
|--------|-----------------|-----------|
| Module 1 | Lecture 1 & 2: Introduction to R | R basics, environment, variables, vectors, lists, matrices, indexing, vector functions (`c()`, `seq()`, `rep()`), matrix operations (`rbind()`, `cbind()`), simple calculations (`sqrt()`, `sum()`, `min()`, `max()`) |
| Module 2 | Lecture 3 & 4: Data Frames & Visualization | Data frames, reading/writing files, descriptive statistics (`mean()`, `median()`, `quantile()`, `summary()`), correlation, visualization (histograms, boxplots, scatterplots, barplots), outlier detection, normality tests |
| Module 3 | Lecture 5 & 6: Statistical Testing | Parametric tests (t-test, paired t-test, ANOVA, post-hoc Tukey/Dunnett), non-parametric tests (Wilcoxon, Kruskal-Wallis), variance tests, checking assumptions, hypothesis testing |
| Project A | Blood Pressure Statistical Analysis | Data exploration, descriptive statistics, visualization, outlier detection, hypothesis testing, linear regression modeling |
| Assignment 1 | Fundamental R Programming & Gene Data Analysis | Vector creation, subsetting, file exploration, gene expression analysis, length-normalized expression, filtering, calculations |
